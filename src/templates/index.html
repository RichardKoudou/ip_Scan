IP_SCAN

Bienvenue sur ip_scan, une petite plateforme qui vous permet de scanner des adresses ip afin d'en recueillir les informations.

Vous avez à votre disposition plusieurs choix de scan. Pour le scan rapide ou le scan tcp udp, vous pouvez chercher une range d'ip comme ceci : 192.168.1.0-254 en sachant que le scan tcp udp est super long. Vous avez ensuite la possibilité de récupérer l'adresse mac en entrant une seule ip

Pour finir le sniff permet d'observer les 50 paquets du réseau. Si vous spécifiez une adresse ip il n'affichera que les requêtes comportant l'adresse ip demandée.
